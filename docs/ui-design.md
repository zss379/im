## 1. 概述

### 1.1 文档目的

本文档定义数莲 PaaS 平台 IM 及 App 基础能力的用户界面与交互设计规范，涵盖 App 移动端与 Web 管理端所有页面的布局结构、交互行为、状态流转及组件定义。基于 PRD 功能需求与 MasterGo 设计稿输出，作为前端开发与 UI 验收的依据。

### 1.2 设计原则

1. **一致性**：多端（iOS/Android/Web）交互模式与视觉风格统一，同一组件在不同端表现一致
2. **高效性**：核心功能操作 ≤ 3 步，会话列表流畅滑动，消息秒级送达
3. **反馈性**：每次操作都有即时反馈（Toast/振动/动画），加载状态骨架屏展示
4. **容错性**：危险操作（删除/解散/退出）二次确认弹窗，弱网/异常状态友好提示

### 1.3 平台差异处理

| 场景 | iOS | Android | Web |
|------|-----|---------|-----|
| 导航 | 底部 Tab + 左侧滑返回 | 底部 Tab + 系统返回键 | 左侧导航 + 浏览器返回 |
| 适配 | 动态岛适配 | 沉浸式状态栏 | 响应式 1280px+ |
| 反馈 | Haptics 轻触反馈 | 振动反馈 | 无 |
| 推送 | APNs | JPush | 浏览器通知 |

### 1.4 通用交互规则

| 交互 | 触发 | 反馈 |
|------|------|------|
| 点击 | 手指/鼠标点击 | 点击态高亮（opacity 0.7），松手恢复 |
| 长按 | 按住 ≥ 500ms | 触感反馈 + 底部操作菜单弹出 |
| 左滑 | 列表项向左滑动 | 露出后方操作按钮 |
| 下拉刷新 | 列表顶部继续下拉 | 加载指示器 + 更新时间标签 |
| 上拉加载 | 滚动到底部 | 底部加载指示器，加载完毕隐藏 |
| 双击 | 图片/视频双击 | 全屏预览切换 |
| 右滑返回 | 屏幕左侧右滑（iOS） | 跟随手势返回上一页 |

## 2. 页面总览与导航架构

### 2.1 App 端页面结构

```
启动页
  │
  ├── 已登录 ──→ 首页（会话列表）
  │                ├── 通讯录
  │                ├── 工作台（现有功能）
  │                └── 我的
  │
  └── 未登录 ──→ 登录页

### 2.2 Web 端页面结构

```
左侧导航栏
  ├── 首页（会话列表）
  │    ├── 聊天窗口
  │    │    └── 群/成员详情（三栏布局）
  │    └── 空状态（未选中会话）
  │
  ├── 通讯录
  │    ├── 组织架构树
  │    └── 成员名片弹窗
  │
  ├── 工作台（现有功能）
  │
  └── 后台管理
       ├── 机器人管理
       ├── 敏感词管理
       ├── 频控配置
       ├── 文件限制配置
       └── 审计日志查询
```

### 2.3 App 底部导航 Tab

| Tab | 图标 | 标签 | 页面内容 |
|-----|------|------|---------|
| 首页 | 消息气泡 | 首页 | 会话列表 |
| 通讯录 | 通讯录 | 通讯录 | 组织架构+我的群组 |
| 工作台 | 网格 | 工作台 | 现有 PaaS 工作台 |

**底部导航栏规格：**
- 高度：iOS 49pt + 安全区域底部，Android 48dp
- 选中态：active 图标 + 品牌色文字
- 未选中态：灰色图标 + 灰色文字
- 未读数：首页 Tab 右上角红点（总未读计数）

### 3.1 页面布局

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│           [品牌 Logo]            │
│                                 │
│        "数莲 · 智慧协作"         │  ← 品牌标语

│                                 │

│                                 │

```

### 3.2 交互规则

| 场景 | 行为 |
|------|------|
| 冷启动 | 展示品牌启动页 ≥ 1s，后台初始化（IM 连接、Token 校验、缓存加载） |
| 已登录且 Token 有效 | 启动页 → 自动跳转首页（会话列表） |
| 未登录/Token 失效 | 启动页 → 自动跳转登录页 |
| 启动资源加载失败 | 展示默认纯色背景，不影响进入下一页面 |

### 3.3 状态说明

| 状态 | 表现 |
|------|------|
| 正常展示 | Logo + 标语 + 版本号 |
| 资源缺失 | 纯品牌色背景，无图片 |
| 后台初始化失败 | 跳转首页后 Toast 提示"部分功能不可用" |

## 4. 登录页

### 4.1 页面布局

```
┌─────────────────────────────────┐
│                                 │
│  ← 返回（仅从设置页进入时显示）    │
│                                 │
│     欢迎回来                     │
│     登录您的账号继续              │
│                                 │
│  ┌─────────────────────┐        │
│  │ 请输入账号           │        │  ← 输入框
│  └─────────────────────┘        │
│                                 │
│  ┌─────────────────────┐        │
│  │ 请输入密码           │  👁    │  ← 输入框 + 密码可见切换
│  └─────────────────────┘        │
│                                 │
│  □ 同意《隐私政策》和《用户协议》  │  ← 隐私政策勾选
│                                 │
│  ┌─────────────────────┐        │
│  │      登 录           │        │  ← 主按钮（置灰/可点击）
│  └─────────────────────┘        │
│                                 │
│  ☑ 记住密码                      │  ← 记住密码开关
│                                 │
└─────────────────────────────────┘
```

### 4.2 交互规则

| 交互 | 行为 |
|------|------|
| 账号/密码为空 | 登录按钮置灰（opacity 0.5，不可点击） |
| 输入框聚焦 | 输入框边框高亮 + placeholder 上浮动画 |
| 密码可见切换 | 点击 👁 图标切换明文/密文显示 |
| 隐私政策勾选 | 未勾选点击登录 → 弹窗提示"请先同意隐私政策" |
| 隐私政策文字 | 点击《隐私政策》→ 跳转 WebView 展示详情 |
| 记住密码 | 开启后本地 Keychain/Keystore 存储加密账号密码 |
| 点击登录 | 按钮显示 loading 转圈，禁用重复点击 |

### 4.3 异常处理

| 场景 | 提示 |
|------|------|
| 账号不存在 | "账号不存在，请检查输入" |
| 密码错误 | "密码错误，还可尝试 4 次" |
| 账号锁定 | "账号已锁定，请 30 分钟后重试"（5 次错误锁定） |
| 网络异常 | "网络连接失败，请检查网络设置" |
| Token 失效（自动跳转） | Toast："登录已过期，请重新登录" |
| 服务器错误 | "服务异常，请稍后重试"（含重试按钮） |

### 4.4 密码错误锁定规则

| 连续错误次数 | 行为 |
|------------|------|
| 1-4 次 | 返回具体错误提示 |
| 第 5 次 | 锁定账号 30 分钟，提示含解锁倒计时 |
| 锁定中 | 提示"账号已锁定，请 30 分钟后重试" |

| 展示项 | 规则 |
|--------|------|
| 头像 | 圆形 44pt，用户头像/群头像，无头像展示末尾文字 |
| 在线标识 | 单聊展示：🟢 在线 / ⚪ 离线 / 🔴 忙碌 / ⭕ 请勿打扰 |

### 5.6 左滑操作

```
┌──────────────┬──────┬──────┬──────┐
│  会话内容区域  │ 标为  │ 置顶  │ 删除  │
│               │ 未读  │      │      │
└──────────────┴──────┴──────┴──────┘
```

| 操作 | 规则 |
|------|------|
| 置顶/取消置顶 | 左滑露出「置顶」按钮（已置顶显示"取消置顶"），最多 5 条，超限弹窗提示 |
| 标为已读/未读 | 左滑露出，已读会话可标为未读恢复红点 |
| 删除 | 左滑到最右露出红色「删除」按钮，点击删除无二次确认（撤销机制：底部 Toast "已删除" + 「撤销」按钮，5s 消失） |

### 5.7 空状态

| 场景 | 展示 |
|------|------|
| 首次使用，无任何会话 | 居中插图 + 文案"暂无聊天，发起你的第一次对话吧" + 「发起聊天」按钮 |
| 已删除全部会话 | 相同空状态，无按钮 |
| 搜索结果为空 | "未找到相关会话" |

### 5.8 状态更新

| 事件 | UI 表现 |
|------|---------|
| 新消息到达（非当前会话） | 会话列表红点数字更新 + 最新消息上移至顶部（动画） |
| 新消息到达（当前会话） | 不更新列表 |
| 消息撤回 | 摘要自动更新为"你撤回了一条消息"或"对方撤回了一条消息" |
| 群名称变更 | 列表群名自动更新 |

### 6.2 顶部导航栏

| 元素 | 交互 |
|------|------|
| ← 返回 | 点击返回会话列表 |
| 会话名称 | 对方姓名（单聊）/ 群名称（群聊） / 机器人名称 |
| 在线状态 | 展示对方在线状态（单聊）/ 群人数（群聊） |
| ⋮ 更多 | 点击弹出右上角菜单（§6.8） |

### 6.3 消息气泡

| 属性 | 己方 | 对方 |
|------|------|------|
| 对齐 | 右对齐 | 左对齐 |
| 气泡颜色 | 品牌蓝色 (#2563EB) 背景 + 白色文字 | 灰色 (#F0F0F0) 背景 + 黑色文字 |
| 圆角 | 大圆角 16pt，右下角小圆角 4pt | 大圆角 16pt，左下角小圆角 4pt |
| 边距 | 右侧距屏幕 12pt | 左侧距屏幕 12pt |
| 连续消息 | 同一人连续多条合并，间距 2pt | 同左 |
| 头像 | 不展示（节省空间） | 展示发送者头像 32pt（群聊） |
| 时间标签 | 首条消息或切换 5 分钟以上显示时间 | 同左 |

### 6.4 消息时间标记规则

| 条件 | 显示 |
|------|------|
| 首条消息 | 显示完整时间 HH:mm |
| 距上一条 ≥ 5 分钟 | 显示时间 HH:mm |
| 距上一条 < 5 分钟 | 不显示时间 |
| 系统消息 | 显示完整时间 |

### 6.5 消息发送状态标识

| 状态 | 图标 | 位置 | 说明 |
|------|------|------|------|
| 发送中 | ⟳ 转圈动画 | 气泡右下角 | 消息暂未到达服务端 |
| 已发送 | ✓ | 气泡右下角 | 服务端已确认接收 |
| 已读（单聊） | ✓✓ 蓝色 | 气泡右下角 | 对方已阅读该消息（单聊专属） |
| 失败 | ⚠ 红色感叹号 | 气泡右下角 | 点击可重发 |

### 6.6 长按消息操作菜单

| 消息类型 | 可用操作 |
|---------|---------|
| 文本 | 复制、引用、转发、多选、撤回（2min内） |
| 图片 | 引用、转发、多选、撤回、保存相册 |
| 视频 | 引用、转发、多选、撤回、保存相册 |
| 文件 | 引用、转发、多选、撤回、保存到手机 |
| 语音 | 引用、转发、撤回（仅自己） |
| 名片 | 查看详情、转发 |
| 卡片 | 查看详情、转发 |
| 系统消息 | 无操作（长按无反应） |

**撤回规则：** 仅 2 分钟内可撤回，超时"撤回"选项置灰并提示"已超过撤回时限"。

### 6.7 输入工具栏

```
